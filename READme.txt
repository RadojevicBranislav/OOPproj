Unesi random zaposlenog - unosi generickog zaposlenog													|
																										|
Nasumicno departmenta - Unos random zaposlenog odredjenog departmentra koriscenjem ComboBoxa			|

Ispisi sve zaposlene - Ispisuje u textArea sve zaposlene												|
																										|
Ispisi troskove firme - Ispisuje ukupan trosak firme koji je utrosen na plate							|
																										|
Obrisi zaposlenog - Uneti id i sifru zapolsenog i ukloniti ga iz liste									|
																										|
Sortiraj po plati - Sortira zaposlene po plati															|
																										|
Ispisi zaposlene sektora - Ispisuje zapolsene odredjenog sektora koji se bira u ComboBox opciji			|
																										|
Ispisi zaposlenog max plata - Ispisuje zaposlenog koji ima najvecu platu								|
																										|
Unesi potrebe - unose se potrebe firme za kadrovima														|
																										|
Generisi potrebe - Nasumicno se postavljaju potrebe														|
																										|
Isprazni formu - Prazni textArea formu																	|
